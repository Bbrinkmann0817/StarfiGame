# Audio

Der Soundtrack wird aktuell **prozedural** im Browser erzeugt
(`src/systems/Audio.js`): entspannter Lo-Fi-Beat beim Erkunden, hektischer
8-Bit-Sound im Quiz-Kampf. Es werden keine externen Dateien benötigt.

Wollt ihr eigene Musik/Effekte nutzen, legt sie hier ab (`.mp3`/`.ogg`/`.wav`)
und sagt Bescheid – dann binde ich einen Datei-basierten Player ein, der den
Generator ersetzt oder ergänzt.

Vorschläge für Dateien:
- `explore.mp3` – Loop für den Erkundungsmodus
- `battle.mp3` – Loop für Quiz-Kämpfe
- `boss.mp3` – Loop für Bosskämpfe
