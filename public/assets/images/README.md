# Bilder & Texturen

Lege hier später eure echten Büro-Fotos und Grafiken ab. Unterstützte Formate: `.png`, `.jpg`, `.webp`.

Aktuell ausgewertete Dateien:

| Datei | Verwendung |
|-------|------------|
| `logo.png` | Wird automatisch auf die Logo-Wand in der Lobby gelegt (transparentes PNG empfohlen, ca. 1500×500). |

Weitere Fotos (z. B. Team-Portraits) können als NPC-Portraits eingebunden werden:
setze dazu in `src/data/npcs.js` beim jeweiligen NPC das Feld `photo` auf z. B.
`'/assets/images/lena.jpg'`.

Fotos als Poster/Bilder an Wänden lassen sich einfach ergänzen – sag einfach Bescheid,
dann verdrahte ich konkrete Plätze.
